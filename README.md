Projeto básico para dashboard com angular

# Instalação
```
git clone https://github.com/walternascimentobarroso/wdashboard.git 

npm install
```

# Módulos
 * Dashboard
 * Perfil
 * Login
 * Usuários

## Gerar Módulos
Módulos, são as paginas da sistema, Ex.: Perfil, Login e etc.
`ng g component pages/my-new-component --spec false`

## Gerar Componentes
Componentes, são os itens reutilizaveis, Ex.: Tabelas, Botões e etc.
`ng g component components/my-new-component --spec false`

## Gerar Serviços
Serviços, são os feramentas para auxiliar nos processos do sistemas, Ex.: Autenticação, CRUD e etc.
`ng g service services/myfolder/my-new-service --spec false`
